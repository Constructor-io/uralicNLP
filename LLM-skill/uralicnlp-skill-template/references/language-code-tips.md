# Language note

- This skill is hard-wired to **Skolt Sami** with the ISO 639-3 code `sms`.
- The CLI does not accept a language argument because it always uses the bundled offline Skolt Sami HFST-OL models.
- The analyzer model is loaded from `scripts/analyser-gt-desc.hfstol`.
- The generator model is loaded from `scripts/generator-gt-norm.hfstol`.
