# Language note

- This skill is hard-wired to **__LANG_NAME__** with the ISO 639-3 code `__LANG_ISO__`.
- The CLI does not accept a language argument because it always uses the bundled offline __LANG_NAME__ HFST-OL models.
- The analyzer model is loaded from `scripts/analyser-gt-desc.hfstol`.
- The generator model is loaded from `scripts/generator-gt-norm.hfstol`.
